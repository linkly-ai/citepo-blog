欢迎来到你的全新博客！这是由 [CitePo](https://citepo.com) 生成的示例文章，帮助你快速上手。

## 什么是 CitePo？

CitePo 是一个轻量级的博客发布平台。你用 **MDX**（Markdown + JSX）写文章，推送到 Git，站点就会自动构建和部署。

<Callout type="tip">
MDX 允许你在 Markdown 中使用丰富的组件。这篇文章本身就是一个 MDX 文件——往下翻到「MDX 组件」部分，看看有哪些可用的组件！
</Callout>

## 写你的第一篇文章

在 `content/` 目录中创建一个新的 `.mdx` 文件。每篇文章顶部需要 **frontmatter**：

```mdx
---
title: 我的第一篇文章
description: 文章的简短描述
date: 2025-01-27
tags: [技术, 教程]
---

你的 Markdown 内容...
```

### 支持的 Frontmatter 字段

| 字段          | 必填 | 说明                                 |
| ------------- | ---- | ------------------------------------ |
| `title`       | 是   | 文章标题                             |
| `description` | 是   | 简短摘要（用于 SEO）                 |
| `date`        | 是   | 发布日期（YYYY-MM-DD）               |
| `tags`        | 否   | 标签数组                             |
| `authors`     | 否   | 作者数组                             |
| `coverImage`  | 否   | 封面图路径（指向 `asset/` 中的图片） |
| `draft`       | 否   | 设为 `true` 则不在生产环境显示       |
