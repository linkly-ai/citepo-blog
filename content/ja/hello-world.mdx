新しいブログへようこそ！これは [CitePo](https://citepo.com) で生成されたサンプル記事です。

## CitePo とは？

CitePo は軽量なブログ公開プラットフォームです。**MDX**（Markdown + JSX）で記事を書き、Git にプッシュすれば、サイトが自動的にビルド・デプロイされます。

<Callout type="tip">
MDX を使うと、Markdown の中でリッチなコンポーネントを使用できます。この記事自体が MDX ファイルです。「MDX コンポーネント」セクションまでスクロールして、使えるコンポーネントをチェックしましょう！
</Callout>

## 最初の記事を書く

`content/` ディレクトリに新しい `.mdx` ファイルを作成してください。各記事の先頭には **frontmatter** が必要です：

```mdx
---
title: 私の最初の記事
description: 記事の簡単な説明
date: 2025-01-27
tags: [技術, チュートリアル]
---

Markdown コンテンツ...
```

### テキストスタイル

**太字**、_イタリック_、~~取り消し線~~、`インラインコード` が使えます。

### コードブロック

```typescript
function greet(name: string): string {
  return `こんにちは、${name}！CitePo へようこそ。`

console.log(greet('世界'))
```

### リスト

- お好みのエディタで執筆（VS Code、Cursor、Vim...）
- `npx citepo dev` でローカルプレビュー
- `npx citepo build` でビルド
- Git にプッシュしてデプロイ

### 引用

### MDX コンポーネント

CitePo には記事を豊かにする組み込み MDX コンポーネントがあります。利用可能なコンポーネントをすべてご紹介します：

#### Callout（コールアウト）

重要な情報を目立たせるために使います：

<Callout type="info">
**info** コールアウト — 補足情報に最適です。
</Callout>

<Callout type="tip">
**tip** コールアウト — 役立つアドバイスを共有できます。
</Callout>

<Callout type="warning">
**warning** コールアウト — 潜在的な問題を読者に警告します。
</Callout>

<Callout type="error">
**error** コールアウト — 重大な問題や破壊的変更を強調します。
</Callout>

#### Accordion（アコーディオン）

FAQ や折りたたみコンテンツに最適：

<AccordionGroup>
  <Accordion title="何かインストールする必要はありますか？">
    いいえ！CitePo は `npx` で動作します。`npx citepo new my-blog` を実行するだけで始められます。グローバルインストールは不要です。
  </Accordion>
  <Accordion title="独自ドメインは使えますか？">
    はい。CitePo はカスタムドメインをサポートしています。デプロイ後、CitePo ダッシュボードでドメインを接続できます。
  </Accordion>
  <Accordion title="CitePo は無料ですか？">
    CitePo は個人ブログ向けの無料プランを提供しています。詳細は [citepo.com](https://citepo.com) をご覧ください。
  </Accordion>
</AccordionGroup>
